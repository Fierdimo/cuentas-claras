# Edge Functions — Referencia

> ⚠️ Los archivos fuente reales están en `supabase/functions/` (raíz del proyecto).
> El CLI de Supabase requiere esa ubicación para `supabase functions deploy`.
>
> Esta carpeta (`src/supabase/functions/`) ya no se usa.
> Edita los archivos en `supabase/functions/` y despliega con:
>
> ```bash
> npx supabase functions deploy store-oauth-token --no-verify-jwt
> npx supabase functions deploy process-invoice --no-verify-jwt
> ```
